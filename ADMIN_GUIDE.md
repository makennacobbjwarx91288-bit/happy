# 管理后台：新增店铺与装修

## 一、新增店铺（新店上架）

1. 登录管理后台，进入 **店铺管理**（Shop Management）。
2. 点击 **新建店铺**。
3. **源码选择**：当前仅提供「胡须店」；后续有服装店、手机店等源码时会在此显示，选对应类型即可。
4. 填写 **店铺名称**、**主域名**（如 `clothing.example.com`），点击 **创建店铺**。
5. 将该域名的 DNS A 记录指向当前服务器（与现有胡须店同一台即可）。
6. 新店订单会出现在同一管理后台的 **数据列表** 中，审核流程与现有店一致（优惠券审核、短信验证等）。  
   **注意**：目前所有店铺共用同一套前台页面与商品数据；若要做「不同店不同页面/商品」，需后续扩展（按店铺加载不同前端或配置）。

---

## 二、装修现有店铺（改名称、主题等）

- **店铺名称、主域名、绑定域名**：在 **店铺管理** 中，对对应店铺卡片进行编辑（改名称）、添加/删除域名即可。
- **店铺展示名、Logo、主题色**：由后台 **店铺管理** 中该店铺的 name、logo_url、theme_color 等决定；若后台已提供编辑入口，直接修改保存即可。
- **页面板块、商品、图片**：当前版本未做「后台可视化装修」，如需改首页板块、商品列表、图片等，需在**项目代码**中修改（如 `src/data/products.ts`、各页面组件），然后重新**推代码并部署**（`git push` + 服务器上 `git pull` 与 `bash update.sh`）。

---

## 三、多店铺与审核

- 多店铺共用**同一套后端、同一套管理后台**；仅店铺身份（域名 / shop_id）不同。
- 客户下单时填写的信息（姓名、地址、电话等）**以客户填写为准**，管理端只做展示与审核，不随管理端语言切换而改变。
