# Theme v2 快速上手

## 目标
让新同学在 10 分钟内完成一次「改样式 -> 改页面文案 -> 发布 -> 回滚」完整流程。

## 操作流程
1. 进入后台 `店铺装修中心 v2`，先选择店铺。
2. 先切换 `手机端 / 电脑端` 和页面标签，确认你正在编辑哪一页。
3. 在 `主题样式` 中修改全局颜色、字体、间距。
4. 在 `页面配置（Page Presets）` 中修改：
   - 集合页 / 商品页 / 支持页 / 公司页
   - 结算页（Checkout）
   - 优惠券页（Coupon Verification）
5. 如需迁移或备份，使用 `复制 JSON / 下载 JSON / 导入 JSON`。
6. 点击 `保存草稿`，前台联调确认后点击 `发布上线`。
7. 若上线异常，在 `版本历史` 中一键 `回滚`。

## 风险控制建议
1. 任何大改动先在草稿完成并截图留档。
2. 发布前至少验证 4 条链路：
   - 首页模块渲染
   - `/checkout` 表单与摘要文案
   - `/verify-coupon` 输入与状态提示
   - 手机端预览
3. 遇到空白页优先检查：
   - 是否启用了 v2
   - JSON 是否被错误导入
   - 最新发布版本是否存在
